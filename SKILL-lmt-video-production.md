## 2. HEYGEN PRODUCTION PROCEDURE

### What HeyGen Does
- Generates Brian's talking head avatar narration
- Supports lower thirds, text overlays, stat animations
- Background color or image behind avatar
- Transitions between scenes
- Does NOT generate illustrated scenes — those come from Canva

### Step-by-Step HeyGen Workflow

**Step 1 — Prepare the script**
- Use plain .txt format only — no markdown, no # headers, no ** bold
- Write spoken lines only — no stage directions in the script field
- Keep apostrophes as straight quotes — HeyGen reads "don't" better than smart quotes
- Break into short paragraphs — one thought per paragraph

**Step 2 — Open HeyGen → New Video**
- Select Brian avatar (custom) or closest professional male avatar, 50+, Black
- Set background: Navy #0E1C2F for open/close sections
- For scene sections: import Canva illustration as background image

**Step 3 — Paste script**
- Paste plain text script into script field
- Review pacing — HeyGen auto-paces but check for rushed sections
- Add manual pauses with [pause] markers where needed

**Step 4 — Add lower thirds**
- Font: DM Sans Bold
- Color: Gold #C8942E text on Navy #0E1C2F background
- Size: 14px
- Position: Lower left
- Timing: Fade in 0.3s · Hold full scene duration · Fade out 0.3s

**Step 5 — Add stat overlays (proof sections)**
- Stats appear one at a time as Brian speaks them
- Gold numbers (#C8942E) · White labels (#FFFFFF)
- Approved stats ONLY: 347+ Pioneers · 3X Completion Rate · 74% More Confident · 23 Organizations
- ⛔ NEVER use: 1,200+ · 85% · 45 sites

**Step 6 — Add end card**
- LMT Logo centered
- learnmoretechnologies.com/join-now/
- "Free. Start Today."
- Trust bar: 347+ Pioneers · 3X Completion Rate · 74% More Confident · 23 Organizations
- Hold 4 seconds · Music resolves

**Step 7 — Export**
- Resolution: 1080p minimum · 4K preferred
- Format: MP4
- Aspect: 16:9 for web/YouTube · Export 9:16 crop separately for social

## 3. CANVA SCENE CREATION PROCEDURE

### What Canva Does for Video
- Generates illustrated scene backgrounds using Canva AI
- Exports scenes as JPG/PNG for import into HeyGen
- Does NOT do video editing or avatar narration

### Step-by-Step Canva Scene Workflow

**Step 1 — Open Canva → Create new design**
- Dimensions: 1920 x 1080px (YouTube/HeyGen standard)
- One design per scene

**Step 2 — Use Canva AI to generate the illustration**
- Click Apps → Text to Image (or AI Image Generator)
- Paste the scene prompt EXACTLY as written in the production brief
- Select style: "Illustration" or "Digital Art" — NOT "Photorealistic"
- Generate 4 options · Select warmest, most dignified result

**Step 3 — Style guidelines for all LMT scenes**
- Warm, humanistic illustration style — NOT cartoon, NOT clipart
- Diverse adults 50+ — Black, Latino, Asian, White — men and women equally
- Real life moments — caught doing something, not posed
- Soft warm lighting in every scene
- ⛔ Never use: generic stock photo style, harsh lighting, condescending expressions

**Step 4 — Export**
- File → Download → JPG (High quality)
- Name file: `lmt-scene-[number]-[description].jpg`
- Example: `lmt-scene-1-job-interview.jpg`

**Step 5 — Import into HeyGen**
- In HeyGen scene where this background is needed
- Upload → Set as background
- Position Brian avatar lower-left or lower-center
- Scene illustration fills remaining space

## 5. LESSON INTRO VIDEO PROCEDURE

Every new LearnDash module needs a lesson intro video. Follow this sequence:

**Step 1 — Write the script (CLO Agent)**
- Runtime: 2:00–2:30 minutes
- Structure: Hook → Promise (3 outcomes) → What's in this module → CTA
- Plain text format for HeyGen
- Voice rules: No "seniors" · No "it's easy" · Adults 50+ always

**Step 2 — Record in HeyGen**
- Brian avatar · Navy background
- No illustrated scenes needed for lesson intros — clean branded look
- Lower thirds: Module number + title in Gold on Navy
- End card: LMT logo + learnmoretechnologies.com/join-now/

**Step 3 — Export and upload**
- Export MP4 1080p
- Upload to YouTube as UNLISTED
- Copy YouTube video ID (11-character string after v= in URL)
- Add to Video ID Registry below

**Step 4 — Embed in LearnDash (CTO Agent)**
- WordPress → LearnDash → Lessons → [lesson name]
- Add YouTube embed block at very top of lesson — before any heading
- Paste URL: https://youtu.be/[VIDEO_ID]?rel=0
- Settings: Autoplay OFF · Captions ON · Related videos OFF
- Below video add italic text: "Watch this first, then work through the lessons below."
- Save → Preview → Confirm live
